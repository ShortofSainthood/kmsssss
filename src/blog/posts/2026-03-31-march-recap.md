I've also been making the most out of my Rubik's cube. I already learned how to solve it a while ago, but I've been working on speedsolving it recently and I've gotten pretty good. ## Site Stuff

### Eleventy is now Build Awesome

For those in IndieWeb spaces, you've probably already heard that [Eleventy is now Build Awesome](https://www.11ty.dev/blog/build-awesome/). A while back [Eleventy became part of Font Awesome](https://www.11ty.dev/blog/eleventy-font-awesome/), and is now being rebranded to better match with other Font Awesome projects and to allow for monetization. Brennan Kenneth Brown has written about this on their blog in ["The End of Eleventy"](https://brennan.day/the-end-of-eleventy/) and ["Build Awesome's Kickstarter is Cancelled"](https://brennan.day/build-awesomes-kickstarter-is-cancelled/). Both posts give a good overview of the situation.

I use Eleventy to make this site, so this is kind of a big deal to me. Some people are already jumping ship to other Static Site Generators. I think I'm going to stick around and see what happens. I've put a lot of work into my site, and I really don't want to go through completely rebuilding it from scratch if I can.

So far, there hasn't been anything *too* egregious. "Build Awesome" hasn't shoehorned in AI, or locked features that were previously free behind a paid tier. All my plugins and scripts work exactly how they did before. Nothing about the actual user experience of using Eleventy has changed. If I didn't follow anyone online, I might not have noticed the rebrand at all. I'm hoping things stay this way, fingers crossed!
