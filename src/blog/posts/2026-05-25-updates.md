It's been a little while since I've last posted. I've been preoccupied with offline stuff, so working on this site hasn't been a priority recently. Still, I've done a few things worth noting, and I really wanted to publicly thank some of the people who were nice enough to take time out of their days to help me out with technical stuff , so here's some updates on that.

## Fanlistings

First off, I've started [my own fanlisting collective](https://fan.sainthood.xyz/) and kicked it off by starting a [fanlisting for Balatro](http://fan.sainthood.xyz/balatro/)! I'll hopefully add more stuff eventually, but this is a good start I think.

I want to give a huge shout out to [Leilukin](https://leilukin.com/)! A while back (like several months ago at this point, wow time flies fast!) they gave me some tech help over email, and recently they made the [source code for their fanlisting collective](https://git.leilukin.com/Leilukin/fanlistings) public, which was an invaluable reference when it came to making my own since I suck at using PHP.

## Site Deployment

Being able to deploy my website has been a priority of mine for a while, but I'm still new to using git and I know basically nothing about actions, plus I'm using a bit of an unconventional set up for static sites (FTP with cPanel instead of GitHub pages or Neocities), so I needed a lot of help.

A *massive* thank you from [Aankhen](https://shivjm.blog/) and [MWDelaney](https://mwdelaney.me/) (who also made [eleventy-github-deployment](https://github.com/MWDelaney/eleventy-github-deployment), the GitHub action I'm using) for helping me out in the [eleventy discord server](https://discord.gg/GBkBy9u) (I hate using discord but the server is still the best place to go for technical help). They were both kind enough to go over error logs with me and help me trouble shoot. I can't thank them enough, I don't think I could have figured it out without them.

## Site Development Slowing Down

I've had a *very* busy April and May, and by the looks of it I'll probably have a pretty busy summer too. As much as I would love to say to expect more site updates now that deployment is working, it'll probably stay the same, or get slower.

I'm not abandoning this site, just reminding everyone that stuff like work, family matters, and my health comes first, and I can't always focus on both those things *and* this site as much as I would like to.
