A super simple one, but one I still see some people struggle with. Modern browsers make pixel art look blurry when scaled up. We can use the CSS image-rendering property to fix this.

No styling

With image-rendering: pixelated;

CSS (Minimum Compatibility)

{% highlight "css" %} img.pixelated { image-rendering: pixelated; } {% endhighlight %}

CSS (Maximum Compatibility)

Also works for older and more obscure browsers

{% highlight "css" %} img.pixelated { image-rendering: crisp-edges; image-rendering: pixelated; image-rendering: -moz-crisp-edges; } {% endhighlight %}
