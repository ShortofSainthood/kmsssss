{% css %}
    .tabwrap section.default {display: block;}
    .tabwrap section {display: none;}
    .tabwrap section:target {display:block;}
    .tabwrap section:target ~ section.default {display:none;}
    section {text-align: center;}
    section h3 {margin: 0 0.5em;}
{% endcss %}

This is a minor edit of [Kalechip’s CSS Tabs](https://kalechips.net/projects/snippets/tabs). I originally made it for [my changelog](/meta/changelog), because I wanted people to be able to filter through different years, but also I didn’t want to deal with JavaScript or PHP. The original has all tab content hidden by default until you click a tab. This edit lets you set a default tab, which is visible by default.

## Demo
<div class="box">
    <p class="speciallink" style="padding: 0.25em 0 1em 0;">
        <a href="#dogs">Dogs</a>
        <a href="#cats">Cats</a>
        <a href="#bunnies">Bunnies</a>
    </p>
    <div class="tabwrap">
        <section id="cats">
            <h3>Cats</h3>
            <p>I like cats too! 🐱</p>
        </section>
        <section id="bunnies">
            <h3>Bunnies</h3>
            <p>I also think bunnies are super cute! 🐰</p>
        </section>
        <section class="default" id="dogs">
            <h3>Dogs</h3>
            <p>I love dogs! 🐶<br>(this is the default tab)</p>
        </section>
    </div>
</div>

## HTML { #html }
{% highlight "html" %}
<!-- Tab links -->
<p><a href="#tab1">Tab #1</a> <a href="#tab2">Tab #2</a></p>

<!-- Tab content -->
<div class="tabwrap">
    <section id="tab2">
        <!--Regular tab content here-->
    </section>
    <section class="default" id="tab1">
        <!--Default tab content here-->
    </section>
</div>
{% endhighlight %}

==You ***must*** put your default tab at the *bottom* of your tabwrap, *after* all of the other tabs.== The CSS will not work properly otherwise. The order of the other tabs doesn't matter. Change "tab1" and "tab2" to whatever you want your tab's names to be.

For accessibility, I highly recommend starting each tab content section with a heading (``<h1>``-``<h6>``).

## CSS { #css }
{% highlight "css" %}
.tabwrap section.default {display: block;}
.tabwrap section {display: none;}
.tabwrap section:target {display:block;}
.tabwrap section:target ~ section.default {display:none;}
{% endhighlight %}
