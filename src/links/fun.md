## Hobbies

### Gaming { #gaming }

Resources related to (non-browser) video games.

* [3D Chao Viewer](https://chao-island.com/viewer/)
* [DarkPattern.games](https://www.darkpattern.games/) - Mobile game review site that reports games with [dark patterns](https://en.wikipedia.org/wiki/Dark_pattern) so you can avoid them.
* [Hacks.guide](https://hacks.guide/) - Includes guides for how to homebrew/mod 3DS, Wii U, PS Vita, Wii, and Nintendo Switch systems.
* The VG Resource - Video Game Assets
    * [The Spriters Resource](https://www.spriters-resource.com/)
